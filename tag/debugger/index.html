    debugger.html 是一个来自于 Mozilla 的现代 JavaScript 调试器， 它是由 React 和 Redux 技术搭建的 web应用。该项目于今年年初开始，是为了取代 Firefox Developer Tools 中目前的调试器。而且，我们也曾想过让一个调试器能调试多个目标并且工作在独立模式下。
